# Project Overview

This project is dedicated to the QDay Prize competition, which aims to break elliptic curve cryptography (ECC) using Shor's algorithm on a quantum computer. The competition challenges participants to demonstrate the capability of quantum computing in cryptanalysis by breaking the largest ECC key possible.

---

## Competition Details

- **Prize**: 1 Bitcoin
- **Deadline**: April 5, 2026
- **Objective**: Break the largest ECC key using Shor's algorithm.

Participants are required to run Shor's algorithm on current quantum hardware, targeting ECC keys of varying lengths. The goal is to demonstrate a general and robust method that scales to larger key sizes without impractical classical computation.

## Shor's Algorithm

Shor's algorithm is a quantum algorithm designed to efficiently solve the problems of integer factorization and discrete logarithms, which underpin the security of classical cryptographic schemes like RSA and ECC. The algorithm demonstrates that a quantum computer can solve these problems exponentially faster than classical algorithms.
